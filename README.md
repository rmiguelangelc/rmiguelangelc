### 📈 Proyecto Destacado

####  [Proyecto_Salud_ML](https://github.com/rmiguelangelc/Proyecto_Salud_ML)
* **Desarrollé** una interfaz web interactiva utilizando la biblioteca **Streamlit** en Python, integrando con éxito el sistema en la nube a través de **Render** para permitir a los usuarios realizar consultas en tiempo real sobre sus perfiles médicos de manera intuitiva y accesible.
* **Diseñé** e implementé un pipeline de Machine Learning utilizando el algoritmo **XGBoost**, el cual procesó la ingesta de datos y el Análisis Exploratorio de Datos (EDA) para automatizar la detección de patrones de salud complejos.
* **Optimicé** el rendimiento del modelo predictivo mediante técnicas avanzadas de ajuste de hiperparámetros, logrando alcanzar una métrica de **85% de precisión (accuracy)** en la evaluación de factores de riesgo para la salud.
* **Estructuré** el control de versiones y el despliegue continuo de todo el ecosistema de código a través de **GitHub**, garantizando la integración limpia de los scripts de ingeniería de características, el entrenamiento del modelo y la disponibilidad de la aplicación web.
